# vue-evernote

## 技术选型

vue 全家桶 + less

## 接口文档

## 运行

首先提前启动[后端服务](github.com/go-study-projs/vue-evernote-api)

```bash
# 安装依赖
yarn install

# 本地调试
yarn run dev

# 生产环境
yarn run build


```
