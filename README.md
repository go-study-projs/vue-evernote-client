一款仿印象笔记的 web app，此为前端代码

## 后端代码
- [Golang版本](https://github.com/go-study-projs/vue-evernote-api)
- [Node版本](https://github.com/ts-study-projs/evernote-server)

## 技术选型

vue 全家桶 + less

## 接口文档

[接口文档](https://github.com/go-study-projs/vue-evernote-api/wiki)



## 运行

首先提前启动后端服务

再启动前端

```bash
# 安装依赖
yarn install

# 本地调试
yarn run dev

# 生产环境
yarn run build


```
